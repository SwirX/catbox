# Feature Ideas for Catbox

- **Script Editor**: Add local variable tracking and type hinting.
- **Rich Text Editor**: Support for custom fonts and color gradients.
- **Image Exporter**: WebAssembly-based processing for faster conversion of large images.
- **Obfuscator**: Control flow flattening and string encryption.
- **CatWeb Asset Manager**: A central hub to track and preview Roblox asset IDs used in projects.
- **Shareable Links**: Generate short URLs to share JSON snippets or API configurations.
